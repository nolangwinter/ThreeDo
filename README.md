<!-- GETTING STARTED -->
## Getting Started


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/nolangwinter/ThreeDo.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the Expo App
   ```sh
   npx expo start
   ```
4. Open in iOS Simulator
   ```sh
    i
   ```
5. Open in Android Simulator
   ```sh
    a
   ```
